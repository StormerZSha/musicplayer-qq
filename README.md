# musicplayer-qq

### 实现的功能
```
1、搜索单曲、专辑、歌手、MV
2、查看专辑、歌手、MV页面
3、热门搜索与搜索记录
4、将音乐添加到播放列表或者移除
5、音乐播放、暂停、上一首、下一首
6、音乐的快进与后退
7、歌词的伪滚动
8、排行榜查看
```
### 未实现的功能
```
音乐进度通过进度条调整
歌词根据内容完美滚动
```
